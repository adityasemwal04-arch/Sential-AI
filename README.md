# Sential AI

A Java-based Smart Helmet Accident Detection and Emergency Alert System.

## Features

- Accident Detection
- GPS Location Tracking
- Emergency SOS Generation
- Google Maps Integration
- Object-Oriented Design
- Maven Project

## Technologies

- Java 17
- Maven
- OOP
- IntelliJ IDEA

## Project Structure

```
src
 └── main
     └── java
         └── com.sentialai
             ├── model
             ├── service
             ├── util
             └── Main.java
```

## How it Works

1. Simulates sensor readings.
2. Detects an accident based on acceleration thresholds.
3. Retrieves GPS coordinates.
4. Generates an emergency report.
5. Creates a Google Maps location link.
